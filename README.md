# Description

MCservControll is code for a discord bot that can turn start/stop/restart you minecraft server via discord.

Your is saved encrypt in an Discord channel that can only be seen by an admin.

This means that if you want to delete or reset your data, you only need to delete the channel and I can't touch your data.

Please let me know your suggestions for improvement. ðŸ’¬

# Commands

/setup -- set up the discord bot

/rolepermission -- set the permission level for a role

/rolecommandpermission -- set the permission level needed for a command

/status -- get the status of your server

/start -- start your server

/stop -- stop your server

/restart -- restart your server

# Install

[Invite the discord bot to your server](https://discord.com/oauth2/authorize?client_id=1472652916010782851&permissions=88246545628176&integration_type=0&scope=bot)

The first command you should run is /setup
